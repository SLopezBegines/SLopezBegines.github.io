# Featured Projects {#projects}

## üè¶ IBEX35 Stock Market Prediction
**Master's Thesis - Machine Learning for Financial Markets**

Comprehensive analysis predicting Spanish stock market movements using machine learning and sentiment analysis. Comparative study of multiple algorithms with rigorous statistical validation.

- **Models**: XGBoost, LightGBM, Random Forest, Neural Networks
- **Data**: Financial indicators (2004-2025) + GDELT sentiment data
- **Validation**: McNemar tests, Diebold-Mariano tests, out-of-sample testing
- **Key Finding**: Sentiment data provides minimal improvement over financial-only models, aligning with efficient market hypothesis
- **Tech Stack**: Python (ML), R (EDA & feature engineering), statistical testing
